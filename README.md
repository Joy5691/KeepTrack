## ğŸ“– Usage

1. **Register/Login**
   - Visit the [live app](https://joy5691.github.io/KeepTrack/)
   - Create an account with email/password
   - Or use it offline (data stored locally)

2. **Add Transactions**
   - Click the "Add" tab
   - Select Income or Expense
   - Enter amount, category, description, and date
   - Submit to track your transaction

3. **View Dashboard**
   - See your total balance at a glance
   - Monitor today's income and expenses
   - Check monthly savings
   - View recent transactions

4. **Check History**
   - Navigate to "History" tab
   - See all transactions chronologically
   - Track spending patterns over time

5. **Analyze Statistics**
   - Open "Stats" tab
   - Review income breakdown by category
   - Analyze expense distribution
   - Understand your financial habits
